# Gemini AI Agent using LangChain

## Project Overview
This project implements an AI Agent using LangChain and Google Gemini LLM.
The agent can understand user queries and use external tools like weather API to provide real-time information.

## Technologies Used
- Python
- LangChain
- LangGraph
- Google Gemini API
- LangChain Tools
- REST API

## Features
- Gemini 2.5 Flash LLM integration
- Tool calling capability
- Weather information retrieval
- Interactive command-line chatbot
